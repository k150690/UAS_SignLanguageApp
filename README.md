SDG yang dipilih:
- SDG 4, Quality Education: Menjamin pendidikan inklusif bagi semua dengan menyediakan alat bantu belajar yang mudah diakses untuk meningkatkan literasi bahasa isyarat.
- SDG 10, Reduced Inequalities: Mengurangi ketimpangan akses komunikasi bagi penyandang disabilitas rungu, menciptakan masyarakat yang lebih inklusif.

Gestura adalah platform edukasi mobile tergamifikasi yang mengubah proses belajar bahasa isyarat (BISINDO) dari metode satu arah konvensional menjadi petualangan belajar yang rekreatif. Menggunakan kombinasi penyimpanan Hybrid (Static Array dan Room Database) untuk persistensi data lokal, aplikasi ini menghubungkan pendapatan koin yang diperoleh pengguna dari hasil menyelesaikan modul materi dan kuis ke sebuah ekosistem ekonomi digital (Item Shop). Koin tersebut dapat dibelanjakan untuk membeli aksesoris (Title) yang akan langsung mengubah identitas visual profil pengguna secara real-time, memberikan rasa pencapaian personal yang berkelanjutan.

Fitur Utama:
- Flashcard Learning Module: Ruang belajar visual interaktif untuk alfabet dan frasa dasar BISINDO yang dilengkapi efek suara penyelesaian menggunakan MediaPlayer.
- Dynamic Quiz Engine: Sistem evaluasi pemahaman pengguna dengan mekanisme pengacakan soal (randomizer), hitung mundur waktu (timer), dan kalkulasi bonus koin dari jawaban combo.
- Token Economy & Direct Purchase Store: Katalog toko lokal yang menjual produk kosmetik digital profil dengan sistem validasi saldo koin otomatis sebelum transaksi diproses.
- Leaderboard: Halaman utama pemantauan profil terintegrasi beserta tabel peringkat kompetitif yang mengurutkan skor pengguna bersama data bot/NPC secara dinamis.

Cara Menjalankan Aplikasi:
Prasyarat (Prerequisites):
- Android Studio (versi terbaru disarankan).
- Android SDK dengan API Level minimal 24 (Android 7.0 Nougat) atau di atasnya.
- Perangkat Android fisik atau Emulator Android yang sudah terkonfigurasi.

Langkah Instalasi & Pengoperasian:
1. Clone Repositori
- Buka terminal atau Git Bash, lalu jalankan perintah berikut:
git clone https://github.com/k150690/UAS_SignLanguageApp.git
2. Buka Proyek di Android Studio
- Buka Android Studio.
- Pilih File -> Open -> Arahkan ke folder hasil clone proyek kalian.
- Tunggu hingga proses Gradle Sync selesai mengunduh seluruh dependensi yang diperlukan.
3. Konfigurasi Perangkat (Jika Menggunakan HP Fisik)
- Aktifkan Developer Options (Opsi Pengembang) di HP Android Anda.
- Aktifkan USB Debugging dan pastikan menu Install via USB sudah diaktifkan (ON).
4. Jalankan Aplikasi (Run)
- Pastikan perangkat emulator atau HP fisik Anda sudah terdeteksi di bagian toolbar atas Android Studio.
- Klik tombol Segitiga Hijau (Run) atau gunakan shortcut Shift + F10.
- Tunggu proses Gradle Build selesai hingga aplikasi otomatis terinstal dan terbuka di layar perangkat Anda.
